Core_Connect_Central

Provided By
Simics Core
Callback Type
void (*)(lang_void *callback_data,
conf_object_t *trigger_obj);
Description
Triggered when Simics has connected to Simics central. If Simics connects to central at initialization, the call will be no earlier than the Core_Initial_Configuration hap.
